Feedback Form Module

A custom Odoo 18 backend module to collect, manage, and analyze customer feedback efficiently.

Key Features

  • Feedback form with rating (1–5)
  • Star-based rating display
  • Automatic priority calculation
  • Kanban view with visual progress indicator
  • List, Form, Graph & Pivot views
  • Auto-generated reference number (FB-0001)
  • Data validation and review workflow
